Promts

The main promt of HTML-Energy is to have fun together. Whether you already have an idea of what you want to create or just want to spend time experimenting with HTML tags and capabilities, it's great!

In any case, we have prepared some promts for you to showcase different ways to interact with HTML. Feel free to use one of them as a reference for your site, or come up with your own for other participants!

404 Poetry

create a 404 error page using HTML and CSS.
You can use images, animations, and funny text to make it engaging.

Working with space

Look around you. What in this space caught your attention? Why?
Try to visualize this object (static or dynamic) and talk about it using HTML.

Prof. Dr. style

Make a personal web-page in Prof. Dr. style

create a 404 error page using HTML and CSS.
You can use images, animations, and funny text to make it engaging.

Digital Workspace

Look at files on desktop, tabs in Browser, old folders names, etc. Are they talking with you? Use it as material to tell a story.

HTML Letter

Write a letter to someone in one of the following cities: NYC, Rotterdam, San Francisco, Toronto, Vancouver, Montreal, Stockholm, London, Saskatoon, Buenos Aires, Stuttgart, Dublin, Austin, Portland (Maine). Include a snippet of handwritten HTML code in the letter. The recipient should render the HTML code by hand on paper and send it back to you.
